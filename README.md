# truecaller.blog

A small application to fetch and read blog posts from truecaller.blog.

## Installation

To start the Server

```
cd /Blog-BE
npm install;
node app.js;

```

To start the front-end application

```
cd /truecaller.blog
npm install
npm run start

```
